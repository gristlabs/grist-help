# Panneau d'administration

Votre installation Grist peut être gérée depuis le Panneau d'administration. Il est disponible pour toutes les instances auto-hébergées de Grist ainsi que pour tous les plans Enterprise, qu'ils soient gérés par Grist Labs ou hébergés sur vos propres serveurs. Le Panneau d'administration n'est disponible que pour l'[administrateur de l'installation](self-managed.md#quest-ce-que-le-compte-administratif) Grist.

Lorsque vous êtes connecté en tant qu'administrateur de votre site, ouvrez le Panneau d'administration soit depuis le menu 'Ressources Grist' en bas à gauche de l'écran, soit depuis le menu déroulant de l'utilisateur.

## Paramètres

La page Installation est l'endroit où vous pouvez configurer les paramètres de l'installation, y compris la [Télémétrie](telemetry.md), les Paramètres de sécurité et le Versioning. Si vous êtes sur un plan Enterprise, vous pourrez également configurer les [Journaux d'audit](install/audit-log-overview.md).

![Panneau d'administration Installation](images/admin-panel/admin-panel-installation.png)

## Contrôles d'administration

Les Contrôles d'administration de Grist fournissent une vue complète des utilisateurs et des ressources dans une installation Grist. Ils sont disponibles sur tous les plans Enterprise, qu'ils soient gérés par Grist Labs ou hébergés sur vos propres serveurs. En savoir plus sur les [Contrôles d'administration](admin-controls.md).
